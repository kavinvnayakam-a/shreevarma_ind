# **App Name**: AyurSphere

## Core Features:

- Product Catalog: Display ayurvedic products from multiple brands with detailed descriptions and images.
- Shopping Cart & Guest Checkout: Allow users to add products to a shopping cart and proceed to checkout as guests, implementing Razorpay for secure payments.
- Customer Profile & Order Tracking: Enable registered customers to manage their profiles and track their order history. Integrate Shiprocket for real-time shipping updates.
- Video Consultation via Twilio: Integrate Twilio for video consultations with ayurvedic practitioners.
- Instant Prescription Generation: After video consultation, generate an instant prescription using AI tool based on the practitioner's input and consultation details.

## Style Guidelines:

- Primary color: Earthy Green (#8FBC8F) to reflect natural and ayurvedic themes.
- Background color: Light Beige (#F5F5DC) for a clean and calming backdrop.
- Accent color: Muted Gold (#DAA520) for highlighting important elements and creating a sense of premium quality.
- Body text: 'PT Sans', a modern and readable sans-serif font suitable for product descriptions and user interface elements.
- Headline font: 'Alegreya', a serif font to lend an elegant and intellectual touch to headings.
- Use simple, line-based icons related to ayurvedic concepts, products, and healthcare for intuitive navigation.
- Implement a clean, responsive grid layout for optimal viewing on all devices, emphasizing product imagery.
- Incorporate subtle transition animations when navigating between pages or adding products to the cart to enhance user experience.
